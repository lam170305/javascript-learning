# JavaScript Learning

Repository lưu trữ quá trình học JavaScript từ cơ bản đến nâng cao.

## Mục tiêu

- Nắm vững JavaScript ES6+
- Thành thạo DOM và Event
- Làm việc với Array, Object, Function
- Hiểu Async JavaScript (Promise, Async/Await)
- Xây dựng các dự án thực tế
- Chuẩn bị cho lộ trình Web Developer & Freelancer
